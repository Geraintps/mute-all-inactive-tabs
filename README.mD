# Mute Inactive Tabs

**Mute Inactive Tabs** is a Firefox extension that makes sure that only your currently open and manually unmuted tabs are unmuted. All inactive and not manually unmuted tabs are kept muted.

The extension now includes a toolbar button and popup where you can toggle **Ignore this site** for the current tab. Ignored sites are never auto-muted when inactive.
